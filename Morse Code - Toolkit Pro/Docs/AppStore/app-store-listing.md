## Description
```
MorseStudio is a full-featured Morse code toolkit built entirely for iPhone.

Type any text and see it converted to Morse code in real time — or go the other way and decode Morse back into plain text. Switch modes instantly with the scope bar at the top.

CONVERTER
• Live bidirectional conversion as you type
• Text → Morse and Morse → Text
• Copy output to clipboard with one tap
• Share via Messages, Mail, or any app
• Auto-saves every conversion to your history

HISTORY
• Every conversion is saved automatically
• Swipe to delete entries you don't need

COMING SOON
• Transmit Morse via flashlight, haptics, or audio tone
• Telegraph key visualizer with real-time LED display
• Practice mode with quizzes and daily challenges
• Favorites — star and search saved conversions

Built with SwiftUI. No ads. No subscriptions. No tracking.
```
